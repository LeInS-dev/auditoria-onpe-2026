# Auditoría ONPE — Dashboard Visual

Dashboard interactivo de auditoría independiente sobre los datos publicados por la
**Oficina Nacional de Procesos Electorales (ONPE)** del Perú para las Elecciones
Generales 2026, con corte al **20 de abril de 2026**.

## ¿Qué muestra?

- Comparación entre los votos oficiales publicados por ONPE y la suma directa de las
  92,699 mesas descargadas vía la API pública.
- Análisis estadístico del bloque de 4,704 "mesas atípicas" con códigos 900,000+ que
  no encajan en el padrón histórico (1—88,000).
- Distribución geográfica y casos puntuales verificables.

## Postura editorial

Este análisis **no afirma manipulación electoral**. Reproduce el método del informe
del 17 de abril 2026 con datos del 20 de abril y muestra qué afirmaciones se
sostienen con datos públicos y cuáles requieren respuesta institucional.

## Fuente

API pública ONPE — endpoint `/presentacion-backend/actas/buscar/mesa`.

## Cómo ver

Abrí `index.html` en cualquier navegador. Funciona offline.
